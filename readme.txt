# 🎰 Bank Robbery Slot - PixiJS Challenge

Projeto desenvolvido como desafio técnico utilizando PixiJS + Vite.

## 🚀 Tecnologias

- PixiJS
- Vite
- JavaScript
- AnimatedSprite
- Responsive Scaling

---

## 🎮 Funcionalidades

- Sistema de reels 6x5
- Animações em sprites
- Raposa animada (Idle)
- Botão Spin funcional
- Responsividade automática
- Sistema de carregamento de assets
- Build otimizado para produção
